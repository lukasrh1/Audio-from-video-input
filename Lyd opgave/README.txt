README

Nu har jeg lavet et program der kan se hvor henne mit hoved er på kameraet
og så bruge den information til at afspille en bestemt lyd.

Wekinator er trænet med mig der sidder foran kameraet i klasseværelset.
På grund af dette kan man nok ikke bruge mit Wekinator program hjemme fra ens ejet værelse eller andre steder.
